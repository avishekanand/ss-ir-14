Features

This mini-project is a skeleton search engine for advanced information retrieval course. The features under investigation are the inverted index, query processing and query enrichment techniques. The modules which are up for the grabs are :

  • Inverted Index Compression
  • Query Processing Techniques
  • Query Expansion and Query Suggestions
  • Retrieval Models
  • Faceted Search
